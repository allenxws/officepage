云安全

又拍云提供全套安全防护解决方案，有效防御各种应用层和网络层的安全攻击和威胁，为用户的业务保驾护航。

服务功能

DDoS 防护

针对目前互联网频发的拒绝服务攻击情况，又拍云采用业界主流的专业 DDoS 防护设备，辅以海量的带宽储备，有效抵御 SYN FLOOD、UDP FLOOD、CC 等高并发的流量型攻击，保障用户业务的正常运行。

WAF 防护

针对互联网应用层的动态数据攻击情况，又拍云提供高性能 WAF 防护引擎，可有效防御 SQL 注入、XSS 攻击、CSRF 攻击、恶意爬虫、扫描器、远程文件包含等黑客攻击，保障用户业务的安全稳定。

全网 HTTPS

支持默认域名 HTTPS 服务和自主域名 HTTPS 服务。用户可上传自定义 SSL 证书，开启自主域名 HTTPS 加密。自定义 SSL 服务默认开启对 SPDY/3.1 的支持，通过对请求进行压缩、优先级排列和多路复用，能有效降低网页的加载时间，实现终端访问“加密+加速”。

全网防盗链

又拍云为用户提供 IP 禁用、域名防盗链、User Agent 防盗链、Token 防盗链、回源鉴权等服务，帮助用户有效控制服务资源的规范访问，减少非正常资源消耗。
