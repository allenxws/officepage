云存储

又拍云提供丰富的 API 接口和多语言 SDK，结合大规模高效云存储集群，为用户提供快速上传、安全可靠的数据存储服务。

服务功能

存储免费

流量用多少，存储免多少。又拍云实现存储减免优惠，以 CDN 流量使用量 1:1 的减免存储空间使用费用。助力用户大幅节省存储成本，让创业更高效。

安全可靠

海量文件存储，弹性扩容，支持多层级 ACL ，实行分布式存储，多数据中心备份，提供高可用和高可靠的数据服务。

融合云存储

为使用云存储服务的用户，额外提供跨平台增量文件同步、镜像迁移、文件上传、无缝切换等功能，提升用户业务的高可用。

灵活上传

提供包括 REST/表单/分块等 API 接口，多种语言的 SDK 包，以及传统 FTP 上传，方便用户选择最合适的方式就近上传，降低云服务使用门槛。
